\section{Introduction}
This report summarizes the experiments performed on the \textbf{SST-5} dataset (5-class sentiment classification) to evaluate \textbf{Counterfactual Gradient Alignment} (CGA).

\section{Experimental Setup}
\textbf{Dataset:} SST-5 (Stanford Sentiment Treebank, 5 classes). \\
\textbf{Model:} BagOfWordsClassifierMultiClass (Embedding Size 32). \\
\textbf{Training:} 5 Epochs, Batch Size 8, Learning Rate 0.0001. \\
\textbf{Variables:}
\begin{itemize}
    \item \textbf{Train Set Size:} \{10, 20, 50, 100, 200\} samples.
    \item \textbf{Mixing Alpha (\(\alpha\)):} \{0.3, 0.5, 0.7\}.
\end{itemize}

\subsection{Observations}
\begin{itemize}
    \item \textbf{Significant Gain at Size 100:} The ReLU directional loss with Alpha 0.3 provided a \textbf{+3.4\%} improvement over baseline.
    \item \textbf{High Complexity:} For 5-class classification, 10-20 samples are extremely sparse. Gains were marginal or zero in these ultra-low data regimes within 5 epochs.
    \item \textbf{Variant Performance:} ReLU and Sign variants showed the best peak performance at intermediate sizes.
\end{itemize}

\section{Conclusion}
CGA demonstrates a meaningful performance increase (+3.4\%) on SST-5 at the 100-sample scale, while maintaining baseline performance at larger scales.
