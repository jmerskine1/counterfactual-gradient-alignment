\section{Introduction}
This report summarizes the experiments performed on the \textbf{SST-5} dataset with \textbf{Active Learning disabled}. 

\section{Experimental Setup}
\textbf{Dataset:} SST-5 (5-class Sentiment). \\
\textbf{Active Learning:} Disabled. \\
\textbf{Variables:}
\begin{itemize}
    \item \textbf{Train Set Size:} \{10, 20, 50, 100, 200\} samples (randomly selected).
    \item \textbf{Mixing Alpha (\(\alpha\)):} \{0.3, 0.5, 0.7\}.
\end{itemize}

\subsection{Observations}
\begin{itemize}
    \item \textbf{Baseline Dominance:} Standard Cross Entropy baseline significantly outperformed directional losses on random subsets.
    \item \textbf{Active Learning Synergy:} Comparing this to the previous SST-5 report (+3.4\% gain with AL), it is evident that CGA relies on high-quality, informative samples to provide a generalization benefit.
\end{itemize}

\section{Conclusion}
On SST-5, CGA does not provide a benefit on randomly selected small datasets. It requires informative sampling (Active Learning) to effectively align gradients.
